{/* Scenario A: Direct Connection Issues */}

Direct Mode (Windows)

Problem: Handshake did not complete

You try to connect on mobile, but the log keeps saying "Handshake did not complete".

Windows Firewall

Your IPs match perfectly (Router & PC are both 192.168.0.56), but Windows is blocking the connection.

  1. Search Windows for "Firewall & Network Protection".
  2. Click on your active network.
  3. Turn Microsoft Defender Firewall OFF temporarily.
  4. Try connecting again.

Oracle Cloud Help

1. Oracle Firewall (VCN)

Go to your Public Subnet Security List and add a rule.

Add INGRESS Rule (Regra de Entrada):
  • Source CIDR: 0.0.0.0/0
  • IP Protocol: UDP (Critical!)
  • Destination Port: 51820

2. Server Diagnosis (SSH)

Error: "Sudo is disabled"?
That means you are in Windows. Reconnect to SSH first!

1. Check Status:

sudo wg show

2. Listen for Signal (TCPDUMP):

sudo tcpdump -i any udp port 51820

If text appears here, your network is PERFECT. The problem is Keys.

3. PLATINUM FIX (Use New Script):

Go to "New Connection" in the app and use the new script. It generates matched keys directly on the server.
